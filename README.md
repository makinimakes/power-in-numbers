# Power in Numbers

An Interdependence Ledger application for financial planning and collaboration.

## Features
-   **Economic Dream Plan:** Calculate your "Thrive" budget.
-   **Project Budgeting:** Estimate project costs based on real needs.
-   **Collaboration:** Invite team members and track equity/shares dynamically.
-   **Data Persistence:** Powered by Supabase.

## Setup
This is a static HTML/JS application.
1.  Open `index.html` (or `login.html`) in your browser to run locally.
2.  Deploy to any static host (Netlify, Vercel, GitHub Pages) to share.

## Tech Stack
-   HTML5, CSS3, Vanilla JavaScript
-   Supabase (Auth & Database)
